## 📊 Sobre el Proyecto
Este análisis integra múltiples fuentes de datos (Ventas, Clientes y Marketing) para ofrecer una visión profunda sobre el rendimiento de campañas y el retorno de inversión (ROI).

**Características destacadas:**
- **Estética Neon Dark Mode**: Diseño futurista de alto impacto visual.
- **Interactividad**: Dashboards desarrollados con Plotly para exploración dinámica de datos.
- **Branding 3D**: Cierre de proyecto con estética profesional "Papiweb Desarrollos Informáticos".
- **Limpieza de Datos Robusta**: Procesamiento y saneamiento automático de datasets.

## ⚡ Cómo usar este Notebook
Para disfrutar de la experiencia completa:
1. Asegúrate de tener instalado `plotly` y `seaborn`.
2. El notebook requiere el archivo `neon_utils.py` (incluido en este repo) para cargar la estética y funciones especiales.
3. Ejecuta todas las celdas para activar las animaciones e interactividad.
